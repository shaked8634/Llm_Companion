# Release 0.0.8

**Release Date:** April 24, 2026

## New Features

- **OpenRouter Support**: Integrated OpenRouter as an LLM provider, giving access to a wide range of models via the OpenRouter API.
- **Custom Provider**: Added support for any OpenAI-compatible API. Users can now configure their own base URL and API key to use custom endpoints.
- **Zoom Synchronization**: The extension UI now automatically syncs its text size with the browser tab's zoom level. Use `CTRL +` / `CTRL -` on the page to adjust the extension's font size proportionally.

## Improvements

- **Error Handling**: Improved provider error messages (e.g., "The provider returned: ...") and implemented automatic error clearing. Previous errors are now cleared from the chat history when a new prompt is initiated.
- **UI Labeling**: Simplified the "Custom" provider label to "Custom" for a cleaner interface.
- **Options UI**: Fixed bugs in the options page that prevented inputting URLs or deleting the last character of API keys.
- **Dependency Updates**: Upgraded core build tools and libraries, including Vite 8, WXT 0.20, and Preact, for better performance and developer experience.

## Maintenance

- **Test Coverage**: Added test coverage reporting using Vitest and `v8`.
- **Testing**: Significantly expanded test coverage with new unit tests for providers and UI components.
- **Developer Guidelines**: Created `AGENTS.md` to provide structured guidance for AI agents and contributors working on the codebase.
- **TypeScript**: Improved type safety and resolved compatibility issues with TypeScript 6.0.
