# Release 0.0.10

**Release Date:** April 24, 2026

## Features

- Added the built-in `Converse with the page` prompt for page-aware questions with typed follow-up input.
- Added the built-in `Translate Selected text` selected-text prompt.

## Improvements

- Updated the About page to show `0.0.0` by default and to display the tagged release version only in built artifacts.
- Added the GitHub mirror link next to the Forgejo repository link in the About page.
- Refined the About page support section so all cryptocurrency donation addresses are grouped inside one shared card.
- Hid selected-text prompts from the main prompt dropdown while keeping them available through the context-menu flow.

## Bug Fixes

- Hardened provider settings handling so missing persisted provider entries no longer crash model discovery or prompt execution.
- Fixed the About page version display so it no longer shows stale hardcoded values in packaged builds.

## Maintenance

- Added regression coverage for version handling, About page links/layout, prompt normalization, selected-text prompt visibility, and page-conversation execution.
- Updated release artifact builds to inject the release tag into `manifest.version_name`.
