# Release 0.0.7

**Release Date:** March 15, 2026

## New Features

- **Selected Text Prompts**: Added support for context menu integration to send selected text directly to LLM Companion. Right-click on any selected text and choose "Send to LLM Companion" to analyze or explain selected content.

- **Improved Ollama Context Detection**: Enhanced model context length detection to properly identify context limits from more fields (`num_ctx`, `context_length`), ensuring better handling of long conversations.

## Improvements

- **Provider State Management**: Fixed provider state management issues and improved UI responsiveness in the options page to prevent flickering from storage updates.

- **UI/UX**: Fixed popup and options page layout overflow and height handling for better user experience.

## Maintenance

- **Renovate Integration**: Added automated dependency updates via Renovate to keep the project up-to-date with latest dependencies.

- **CI/CD**: Added Chrome Web Store upload and publish steps to the release workflow.

- **Screenshots**: Added screenshots to the project for documentation and store listings.
