# Release 0.0.9

**Release Date:** April 24, 2026

## Bug Fixes

- Fixed chat streaming timeout handling to avoid stalled requests and incorrect error reporting.
- Improved runtime error handling so internal implementation details are no longer surfaced in the UI.
- Fixed the model selector flow for first-time installs when only OpenRouter is enabled.
- Disabled Ollama by default on fresh installs so startup model discovery no longer fails against a missing local Ollama instance.

## Improvements

- Improved provider discovery and model refresh behavior for OpenRouter-only setups.
- Added coverage for provider details, model discovery, store defaults, popup behavior, and provider-selection UI flows.
- Updated contributor guidance in `AGENTS.md` with development and release workflow instructions.

## Maintenance

- Renamed the custom provider implementation file to `src/lib/providers/custom.ts` and kept provider wiring aligned across the codebase.
- Prepared the repository for the `0.0.9` release tag and release artifact workflow.
